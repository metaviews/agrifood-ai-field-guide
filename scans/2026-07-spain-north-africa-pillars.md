**Status.** Two-phase cycle (scout + deep). Builds on:
- `scans/2026-07-mena-scan.md` (MENA + Lebanon scan) for regional cycle-pattern parallel

This scan is the cycle-scan for **Spain + North Africa** (a *Maghreb / Mediterranean-region / Iberian* geographic-cluster-pattern cycle rather than a regional-cycle-only). Spain is in the *EU-cluster-pattern context* with corporate-vendor-deployment sub-pattern; North Africa (Morocco + Tunisia primarily) is in the *MENA-region-context* with state-corporate + startup-emerging sub-patterns. The substantive analytical claim: *EU-cluster-pattern is layered-mix (cooperative-governance + state-trade-promotion + accelerator + corporate-vendor-deployment + state-corporate); MENA-region cluster is layered-mix (Israeli venture-funded + UAE state-standards-setter + UAE deployment-of-record + Lebanon startup-ecosystem + Morocco state-corporate + Tunisia startup-emerging-expansion)*.

**Scope.** Spain (Iberian Peninsula, EU member); North Africa (Morocco + Tunisia + Algeria + Libya + Egypt-North-Africa-context); Mediterranean-connection (Spain + North Africa geographically; potential cluster-pattern connections).

**Cutoff.** 2026-07-19.

**Why this scan now.** The user-requested Spain + North Africa cycle explicitly tests *two-region-cluster-pattern* rather than single-region cluster pattern. The prior MENA cycle's *three-sub-pattern observation* (Israeli venture-funded / UAE state-strategy dual-mode / Lebanon startup-ecosystem) gains a fourth / fifth / sixth geographic-cluster pattern with Morocco state-corporate and Tunisia startup-emerging-expansion. Spain at the EU-cluster-pattern node provides corpus's first explicit EU-cluster-pattern sub-pattern articulation beyond the existing Dutch cooperative-governance anchor.

**Method.** Two-phase: scout first (six parallel searches across Spanish agritech corporations; Spanish national/EU programmes; Moroccan agritech; Tunisian/Algerian/Egyptian agritech; regional institutional layer; critical-voice); then deep-cycle drafting (4 units + 1 integrating scan). Each unit gets tier-1 verification where available; primary-source verification at scout-phase before deep drafting.

## 2. Deep-cycle drafts — four units drafted

The deep-cycle phase drafted four substantive units:

### 2.1 Spain — Ekonoke (corporate-vendor-deployment within EU-cluster-pattern context)

**Anchor unit:** `units/ekonoke-spanish-indoor-hop-hydroponics-ai.md`

Ekonoke is corpus's first primary-source-documented Spanish-origin agritech AI deployment at tier-1. Spanish startup Ekonoke cultivates hops hydroponically indoors, **95% less water than open-field hops** (per Reuters April 2023 + Sifted 2025). Almost 10 metres high indoor vertical cultivation with multi-colour LED + renewable-powered hydroponic system. Strategic motivation: climate-adaptation substrate for hops otherwise climate-threatened in traditional growing regions. "Save climate-threatened beer" framing.

**Substantive cluster-pattern observation:**

- 95% water-savings is corpus's most extreme water-savings figure to date (vs. Lebanon Ground Vertical Farming 90% via recirculation; vs. Kilimo Argentina 13% irrigation reduction)
- Spanish agritech operates at EU-cluster-pattern context with corporate-vendor-deployment shape distinct from NA-equipment-vendor + cooperative-governance-NL
- Climate-adaptation-driven deployment shape (vs. corporate-cost-savings-driven; vs. state-mandate-driven) is corpus's first explicit climate-adaptation-driver framing

### 2.2 Spain — AgriFoodTech 2025 ecosystem (state-trade-promotion + accelerator + AI-strategy)

**Anchor unit:** `units/spain-agrifoodtech-2025-ecosystem-eatable-adventures.md`

Spanish AgriFoodTech 2025 ecosystem at the aggregate level:

- **416 active Spanish AgriFoodTech startups** (5% YoY growth)
- **48% AI adoption rate** within Spanish ecosystem (corpus-distinctive: highest ecosystem-level AI-adoption metric to date)
- **€123M investment in 2025** (down 31.3% YoY; consolidating rather than contracting)
- Eatable Adventures + ICEX + ENIA + CIIAA + IIIA-CSIC + AGRARIA Project + GMV as institutional substrate
- Eatable Adventures multi-city reach (Madrid / Barcelona / Verona / Bogotá / London / San Francisco / Montevideo)

**Substantive cluster-pattern observation:**

- Spanish ecosystem is corpus's first *EU-cluster-pattern-with-corporate-vendor-deployment* sub-pattern observation
- EU-cluster-pattern is now corpus-distinct as layered-mix: cooperative-governance (NL/DE via JoinData) + state-trade-promotion + accelerator + AI-strategy (Spain) + corporate-vendor-deployment
- 31.3% YoY investment decline + 48% AI-adoption + 5% YoY startup-count-growth = *ecosystem-consolidation* pattern; corpus's first deep-cycle such observation
- F&A Next 2026 exhibitor list documents six named Spanish innovators (ClicRecycle, Bio2Coat, Néboda, IOVI, Fliwer, Ingredalia)

### 2.3 Tunisia — RoboCare (startup-ecosystem-emerging-expansion layer)

**Anchor unit:** `units/tunisia-robocare-precision-agriculture.md`

Tunisian Sfax-founded RoboCare is the corpus's first primary-source-documented Tunisian agritech AI deployment. Multi-source data integration (satellite + drone + IoT + weather + field expertise) through AI models for early disease/stress detection, resource optimization, and farm performance. June 2026: 216 Capital six-figure investment; African + Middle Eastern expansion scope.

**Substantive cluster-pattern observation:**

- Tunisian RoboCare fills the *Maghreb + North-Africa-extension geographic-cluster gap* (cycle 4 MENA established UAE / Lebanon / Israel; Tunisia + Morocco were stated gaps)
- 216 Capital's *regional-African venture strategy* is corpus's first explicit African-and-MENA regional-VC observation; distinct from global-VC-strategy
- RoboCare multi-source data integration (satellite + drone + IoT + weather + field expertise) is corpus's most multi-source-integration deployment at startup-ecosystem scale
- Multilingual deployment shape (Arabic + French + English) is corpus's first Maghreb-multilingual-scope observation

### 2.4 Morocco — Al Moutmir (state-corporate + state-strategy hybrid)

**Anchor unit:** `units/morocco-al-moutmir-ocp-agritech.md`

Al Moutmir is corpus's first primary-source-documented Moroccan agritech AI deployment. OCP Group's multi-service agritech program launched September 2018; Integrated Crop Program (ICP) framework with four pillars (soil-test-based fertilization, soil testing, crop-specific protocol, farmer training); AI-assisted fertilizer recommendation + smart irrigation. Strategic context: OCP Group's structural significance (world's-largest-phosphate + state-affiliated + global-scale food-system); Green Generation 2020-2030 state strategy; SIAM 2026 confirmed deployment; 2025-2026 bilan.

**Substantive cluster-pattern observation:**

- Al Moutmir fills the *Morocco + Maghreb-state-strategy* geographic-cluster gap
- OCP Group's state-affiliated-phosphate substrate is corpus's most-distinctive *state-corporate + AI-fertilizer-recommendation* deployment category
- Green Generation 2020-2030 is corpus's first explicit Maghreb-region state-strategy at the agritech-AI tier
- Multilingual deployment shape (Arabic + French + Berber) is corpus's first Maghreb multilingual-scope observation
- Al Moutmir + UAE ADAFSA together form corpus's first state-strategy pattern comparison: *state-corporate + product-deployment-led* (Morocco) vs. *state-federal + standards-led* (UAE) — substantive structural distinction

## 3. Three-tier cluster-pattern observation across the cycle

This cycle surfaces **three cluster-pattern tier observations** operationally observed across Spain + North Africa:

### 3.1 EU-cluster-pattern sub-pattern: state-trade-promotion + accelerator + AI-strategy (Spain)

**Driver:** State-trade-promotion (ICEX) + accelerator (Eatable Adventures) + AI-strategy (ENIA) + CIIAA chair + IIIA-CSIC

**Substrate:** 416 startups; 48% AI adoption; €123M 2025 investment; multi-city international accelerator reach

**Substantive sub-pattern observation:** EU-cluster-pattern is layered-mix. The Spanish approach is corporate-vendor-deployment within EU-cluster-pattern-context — distinct from the Dutch/German cooperative-governance approach (JoinData / cooperative tradition). Both operate under broader EU institutional framework.

### 3.2 Mediterranean-sub-region cluster (Spain + North Africa)

**Driver:** Multiple:
- Spain: state-trade-promotion + accelerator + AI-strategy + corporate-vendor-deployment (Ekonoke)
- Morocco: state-corporate (OCP) + state-strategy (Green Generation 2020-2030)
- Tunisia: startup-ecosystem-emerging-expansion (RoboCare + 216 Capital)
- Algeria / Libya / Egypt: future-cycle gap (G-152, G-153)

**Substrate:** Mediterranean-connection geography; multilingual deployment shape (Arabic + French + Berber + Spanish + Catalan + Basque + Galician)

**Substantive sub-pattern observation:** Mediterranean-sub-region cluster operates across two regional-cluster-pattern nodes (EU-cluster-pattern in Spain; MENA-region cluster in North Africa). The *Mediterranean* geography bridges the two regional nodes. Worth surfacing as a substantive cross-regional cluster-pattern observation.

### 3.3 MENA-region cluster five-cluster-pattern (Israeli + UAE + Lebanese + Moroccan + Tunisian)

**Driver:** Multiple, distinct sub-patterns:
- Israeli venture-funded-mature
- UAE state-federal-standards + state-deployment-of-record dual-mode
- Lebanese startup-ecosystem-driven
- Moroccan state-corporate + state-strategy hybrid
- Tunisian startup-ecosystem-emerging-expansion

**Substrate:** MENA-region geographic cluster; multilingual deployment scope (Arabic + French + Berber + Hebrew + English)

**Substantive cluster-pattern observation:** The cycle 4 MENA three-sub-pattern observation is now refined to **five-cluster-pattern** observation with the addition of Moroccan state-corporate and Tunisian startup-emerging-expansion.
