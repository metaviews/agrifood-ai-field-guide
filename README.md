# Agrifood AI Field Guide

A knowledge base and presentation framework for public education on artificial intelligence across the agrifood sector.

**Start here:** [VISION.md](VISION.md) — the framing, audience, stance, taxonomy, freshness rules, and positioning relative to existing work.

## Layout

- `VISION.md` — the project's framing and operating principles
- `scans/` — environmental scan outputs (incoming)
- `scans/` — environmental scan outputs (10 cycles through July 2026: initial, regional, regional-industry-NA-EU, Canada-FCC deepening, France, Quebec, China-deepening, data cycle, LAC, open source, US academic research)
- `units/` — content units, the substrate from which talks are built (60+ units across vendor deployments, US/EU/Canadian academic institutes, data cooperatives, IDSov frameworks, etc.)
- `quotes/` — speaker voices (verbatim quotes + paraphrased positions) for use in talks (24 entries across 5 categories)
- `talks/` — methodology layer: five working archetypes (vendor-sweep / data-sovereignty / adoption-diagnosis / cooperative-alternative / critical-lens) for assembling the library into talks. See `talks/README.md` for operating principles.
- `LICENSE` — CC BY 4.0 for content
- `NOTICE` — MIT for code
