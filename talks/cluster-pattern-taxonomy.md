# Cluster-Pattern Taxonomy — How countries shape their agrifood AI cluster

**Status.** Promoted from `talks/archetypes/06-regional-cluster-comparison.md` because six regional deepening cycles (LAC, Brazilian beef + Chile-Canada, Argentine beef + Brazilian seed, MENA + Lebanon, Spain + North Africa, Spanish cooperatives) surfaced a cluster-pattern vocabulary that has outgrown its host document. The substantive finding that emerged across cycles — *cluster-patterns are plural; cluster patterns layer; some cluster patterns (cluster-with-three-structures) repeat across regions with structurally-distinct substrates* — needs a canonical home separate from any single archetype.

**Why this exists.** The field guide's regional-cluster comparison asks: *do countries with similar demographic imperatives (ageing-farm populations; rural labour decline; food-security anxiety) operationalise AI differently?* The answer across the corpus's six regional cycles: yes — and the operational shape of the cluster produces different outcomes for *who controls the data*, *who captures the value*, *what counts as deployment-of-record*, and *what's the deployment-shape critique*.

**How to use this doc.** Treat the cluster-pattern table as a query target, not a list. When building a regional talk, query the row that matches your region and follow the cross-references into units + scans + contested claims. When building a comparative talk across regions, query the cluster-pattern-name column to find all regions that operate the same pattern.

## 1. Cluster-pattern name vocabulary

| Pattern name | Definition |
|---|---|
| **NA-equipment-vendor** | Agricultural AI deployment led by industrial-machinery majors (tractors, drones, sensors) operating proprietary vendor platforms. Most visible at scale in NA-US; replicated where global majors sell. |
| **Cooperative-governance (NL pattern)** | Non-profit cooperative data-rights infrastructure where the data and benefits flow back to the farmer. Cultural substrate required: cooperative tradition (Netherlands), Federal-state + cooperative-bank presence. |
| **State-vendor hybrid + provincial autonomy** | State-led policy framing + vendor autonomy at the operational level + provincial-level variation. Substrate: state-aligned with vendor autonomy operationally; provinces drive variation. |
| **State-DPI substrate + private vendor layers** | National state-developed digital public infrastructure (DPI) as substrate (farmer registry, plot registry, crop-sown registry; e-Mitra chatbot); private vendor layers build on top. Cascade across state agriculture departments. |
| **Equipment-vendor industrial automation** | Industrial-machinery majors (motorcycle engines, boats, robotics) apply competence to agriculture. Substrate: industrial-machinery vendor concentration; state-side regulatory framing. |
| **State-anchored cluster programme** | State-led cluster anchoring + multi-site innovation valleys + multi-vendor participation; vendor participation secondary to state anchorage. Strong quantitative policy levers. |
| **Multilateral-institutional convening (LAC pattern)** | IICA + IDB + CAF + Bayer + PROCISUR + U of Córdoba + AWS as convening substrate. Distinct from state-DPI substrate. |
| **Israeli venture-funded agritech-startup cluster** | Mature Israeli national ecosystem (750+ companies; 150+ startups; 33 funding rounds >$1M annually); venture-funded corporate-vendor deployment. |
| **State-strategy dual-mode** | State federal agency operating both (a) standards-setter mode (institutional governance framework / international-standards certification) and (b) deployment-of-record mode (specific AI deployment). Cultural-stewardship dimension distinct. |
| **State-corporate + state-strategy hybrid** | Multinational state-affiliated corporation (world's largest phosphate / world's largest fertilizer / etc.) operating AI deployment within national state strategy. |
| **Startup-ecosystem-emerging-expansion** | Single-agri startup with regional-VC funding; multi-source-data integration; expansion scope to regional + sub-continental scope. |
| **Cooperative-federation-institutional-substrate + emerging-single-coop deployment** | Multi-sector cooperative-federation (worker cooperatives since 1956; €11B+ revenue; 70,000+ workers) + emerging individual-cooperative AI deployment on open-innovation substrate. |
| **Cross-border cluster pattern** | Multi-country cluster spanning counter-season production + genetic-tooling flow + multinational-corporate-pipelined deployment. |

## 3. Cross-region cluster-pattern observations

Three observations have emerged from six regional cycles:

### 3.1 Cluster patterns are layered-mix, not single-pattern

The LAC, MENA, and EU-cluster-pattern rows all observe *layered-mix* structure: multiple operative cluster patterns operating within the same regional frame. *Three-region observation of layered-mix structure*.

### 3.2 Cluster-with-three-structures is a cross-region cluster pattern (substantive finding)

Two regional cases observe the cluster-with-three-structures pattern:

| Tier | Brazilian seed AI | Spanish cooperative AI |
|---|---|---|
| Tier-1: peer-reviewed-research-led | Sangjan 2025 cited 24; Tedeschi 2025 PMC cited 20; Stupar 2024 PMC | García-Lafuente 2026 BFJ; Ciruela-Lorenzo 2020 cited 248; Santos 2024 cited 72; Arévalo-Royo 2025 cited 25; Sadjadi 2023 cited 81 |
| **Tier-2** | **Multinational-corporate-pipelined** (Bayer / Syngenta / BASF / Corteva Brazil) | **Mondragón institutional-federation-anchor** (federation of worker cooperatives since 1956; €11.213B 2024; 70,000+ workers) |
| Tier-3: substantially-empty at deployment-of-record | Empty at Brazilian-origin-corporate-vendor | Emerging-single-coop deployment (COVAP Reto Innovación Abierta 2025) + Andalusian Cooperativas Agroalimentarias |

*The same cluster-with-three-structures pattern operates across two regional contexts with structurally-distinct Tier-2 substrates.* This is corpus-valuable; the next-cycle question is whether the same pattern operates across additional regional contexts (Italian agritech, Egyptian agritech, etc.).

### 3.3 Cluster-with-state-substrate as substantive pattern

Argentine beef AI (SENASA + SIGSA + World Bank financing) is the corpus's first cluster-with-state-substrate example distinct from cluster-with-state-strategy (UAE ADAFSA dual-mode). Worth surfacing in any talk framing state-led agricultural AI — *state-driven* and *state-strategy* are *not* synonyms.
