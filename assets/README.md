# Assets — visual identity

This directory holds visual-identity assets for the project. The goat is a *visual identity element only*. It is not referenced in cycles, commits, analysis, or framework discussions.

## Design intent

The mountain goat on a rocky outcrop looking out over an agricultural valley evokes:

- **Diverse grazing method.** Goats eat a wide range of vegetation — grass, flowers, vines, trees. The metaphor evokes range and openness rather than focus or specialisation.
- **Overview without verdict.** The goat observes the valley; it does not declare, survey, or judge. The valley below is the agrifood substrate the project documents.
- **Quiet observation, not heroic declaration.** The composition is contemplative. The goat is at the edge of the tree line, not at the apex of a mountain.

## Constraints

- The goat is a **visual identity element only**. It does not appear in cycle annotations, commit messages, analysis units, framework units, or substantive talk content.
- The goat is **graphical offset to academic prose**, not a mascot character. It does not narrate, does not speak, does not recur with a personality.
- The goat does **not appear in unit files, scan files, or quote files**. Content documents stand on their substance; the goat is project voice, not content voice.
- Placement, where used, is constrained to: README headers, talk archetype files, the cluster-pattern taxonomy document, and (when built) deck-generation tooling output.

## Status

All four files are v0 placeholders generated as starting points for the visual identity. None has been promoted to canonical logo status. Final canonical asset will likely be:

- a vectorised SVG version of the chosen composition (PNG is a starting point for review)
- in a single colour (sepia-on-cream) at two scales (small mark / full mark)
- optionally with a single-character or single-word variant for small-format use (favicon, social-media avatar)

Replacement criteria:

- Vector over raster (SVG preferred for final asset)
- Consistent line weight at all scales
- Legible at favicon size (32px)
- Composition that does not depend on a specific valley layout (regional flexibility)
